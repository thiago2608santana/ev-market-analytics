# EV Market Analytics

Este repositório contém uma análise de dados focada no mercado de veículos elétricos, utilizando o conjunto de dados **Electric Vehicle Market and Pricing Dataset 2026**. O principal objetivo deste projeto é explorar, analisar e aplicar métodos de tratamento de dados para entender melhor as características, preços e o cenário do mercado de EVs.

## 📂 Estrutura do Projeto

- **`Dados/`**: Diretório que armazena os dados brutos utilizados na análise.
  - `ev_market_2026.csv`: Dataset principal contendo informações detalhadas sobre os veículos elétricos.
- **`aula_metodos_tratamento_dados_30_04_2026.ipynb`**: Jupyter Notebook contendo o passo a passo da análise exploratória de dados (EDA) inicial, incluindo verificação de valores nulos, estruturação das colunas e métodos de tratamento de dados utilizando a biblioteca `pandas`.

## 🚀 Tecnologias Utilizadas

- **Python**
- **Pandas** (para manipulação e tratamento de dados)
- **Jupyter Notebook / Google Colab**

## 📊 Fonte de Dados

Os dados utilizados neste projeto foram extraídos da plataforma Kaggle:
🔗 [Electric Vehicle Market and Pricing Dataset 2026](https://www.kaggle.com/datasets/patelris/electric-vehicle-market-and-pricing-dataset-2026)

## ⚙️ Como executar localmente

1. Certifique-se de ter o Python e o Jupyter instalados em sua máquina.
2. Clone o repositório:
   ```bash
   git clone https://github.com/thiago2608santana/ev-market-analytics.git
   ```
3. Instale a biblioteca Pandas:
   ```bash
   pip install pandas
   ```
4. Inicie o Jupyter Notebook e abra o arquivo correspondente:
   ```bash
   jupyter notebook aula_metodos_tratamento_dados_30_04_2026.ipynb
   ```
